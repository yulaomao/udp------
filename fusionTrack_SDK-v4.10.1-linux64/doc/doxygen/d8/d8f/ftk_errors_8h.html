Enumeration Type Documentation

enum ftkError : int32
strong

Error codes.

Negative values are used for warnings (i.e. non-fatal issues), whilst positive values are used for errors. An error should stop the normal execution, as the data may be corrupted, missing, old, etc.

Enumerator
FTK_WAR_DISABLED 

Status when an available feature was disabled by the user.

This is for instance returned when the user tries to read the acceleration value when they disabled the periodic sending of the accelerometer data.
