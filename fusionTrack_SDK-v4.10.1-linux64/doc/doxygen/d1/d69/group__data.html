ftkError ftkGetAccelerometerData ( ftkLibrary  lib,
uint64  sn,
uint32  index,
ftk3DPoint *  value 
)

Getter for the accelerometer data.

This function allows to access the current accelerometer data. The acceleration are returned as a 3D vector in standard units.

Warning
This function takes some time to be completed, it should not be called from the acquisition loop.
Parameters
[in]liban initialised library handle.
[in]sna valid serial number of the device.
[in]indexindex of the accelerometer to read.
[out]valuepointer on the output acceleration read by the first accelerometer in units.
Return values
ftkError::FTK_OKif the data could be retrieved successfully,
ftkError::FTK_WAR_NOT_SUPPORTEDif the fusionTrack does not support that feature,
ftkError::FTK_WAR_DISABLEDis the periodic sending of the accelerometer data has been disabled,
ftkError::FTK_ERR_INV_PTRthe lib handle was not correctly initialised, or the value pointer is nullptr,
ftkError::FTK_ERR_INV_INDEXif the given index is invalid,
ftkError::FTK_ERR_INV_SNif the device could not be retrieved,
ftkError::FTK_ERR_INV_OPT_VALif the accelerometer status indicates the reading is invalid,
ftkError::FTK_ERR_READif the wanted register cannot be read.
ftkError ftkGetRealTimeClock ( ftkLibrary  lib,
uint64  sn,
uint64 *  timestamp 
)

Getter for the real time clock timestamp.

This function allows to get the current timestamp of the fusionTrack, given by the on board real time clock module. The value is the linux timestamp: the number of seconds since January 1st 1970 at 00:00:00.

Warning
This function takes some time to be completed, it should not be called from the acquisition loop.
Parameters
[in]liban initialised library handle.
[in]sna valid serial number of the device.
[out]timestamppointer on the written timestamp.
Return values
ftkError::FTK_OKif the data could be retrieved successfully.
ftkError::FTK_ERR_INV_PTRthe lib handle was not correctly initialised, or the firstValue pointer is null,
ftkError::FTK_ERR_INV_SNif the device could not be retrieved,
ftkError::FTK_ERR_INV_OPT_PARif the wanted register cannot be read,
ftkError::FTK_WAR_NOT_SUPPORTEDif the RTC component cannot be accessed (occurs for firmware versions prior to 1.1.6.6B).
