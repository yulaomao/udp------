Detailed Description

Class holding the information for a 64-fiducials marker.

This class contains all the information about a 64-fiducials marker. The instance does know if it has been properly set (i.e. whether its member values make sense).

The instance contains copies of all FiducialData instances used to reconstruct the marker.
