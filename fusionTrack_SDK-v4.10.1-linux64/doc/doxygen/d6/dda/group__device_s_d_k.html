Opens the XML file for the dump.

This function opens a XML file in which the data will be dumped. If a file is already opened, an error is returned.

Parameters
[in]libinitialised library handle.
[in]serialNbrserial number of the device for which the file is created.
[in]fileNamename of the XML file to create (if not provided, then a file Dump_serialNumber.xml is created).
Return values
ftkError::FTK_OKif the file could be successfully opened and written,
ftkError::FTK_ERR_INV_PTRif lib is nullptr,
ftkError::FTK_ERR_INV_SNif serialNbr is not a valid serial number,
ftkError::FTK_ERR_WRITEif a file is already opened or if if the file cannot be opened.
