C++ API
Warning
The C++ API is not yet fully validated.

Some functionalities of the C-style SDK are not implemented yet:

  • data dumping;
  • frame reprocessing.

The C++ API consists of a set of classes, which wrap the C-style SDK. It is provided as source and can be freely modified by the user. Its goal is to provide a nice and clean interface to be used in C++ programs. All the classes and functions lie in the atracsys namespace.

Detecting the connected devices and get the serial number of the last detected one reads:

#include <TrackingSystem.hpp>
#include <iomanip>
#include <iostream>
#include <vector>
using namespace std;
int main()
{
atracsys::TrackingSystem trSystem();
atracsys::Status retCode( trSystem.initialise() );
if ( retCode != atracsys::Status::Ok )
{
trSystem.streamLastError();
return 1;
}
retCode = trSystem.enumerateDevices();
if ( retCode != atracsys::Status::Ok )
{
trSystem.streamLastError();
return 1;
}
vector< atracsys::DeviceInfo > devices;
retCode = trSystem.getEnumeratedDevices( devices );
if ( retCode != atracsys::Status::Ok )
{
trSystem.streamLastError();
return 1;
}
cout << "Detected device " << hex << showbase
<< devices.back().serialNumber() << endl;
return 0;
}
