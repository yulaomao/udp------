Atracsys Passive Tracking SDK Documentation

Description

This library enables to acquire high-level tracking data of either an infiniTrack, a fusionTrack or a spryTrack tracking system.

Functions are available through a comprehensible C-style API shared library, which can be used directly or via an open source C++ API.

Functions of this SDK are well documented including comprehensive examples. Alternatively, a first step tutorial can be found in the Tutorial page. Samples programs are also a good starting point to understand how to integrate this SDK in your project.
