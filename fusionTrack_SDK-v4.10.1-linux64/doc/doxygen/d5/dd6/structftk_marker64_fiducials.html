Detailed Description

Marker data after left-right triangulation and marker reconstruction, with up to 64 fiducials.

This structure is the equivalent of ftkMarker, but handling up to 64 fiducials. The differences are

  • ftkMarker64Fiducials::geometryPresenceMask is a uint64;
  • ftkMarker64Fiducials::fiducialCorresp is an array of 64 uint32.
